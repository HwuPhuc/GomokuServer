# Gomoku Server
An online Gomoku game created by using Java.

The aim is to understand how to make an online game where players can connect to each other through Internet and play together.

Make sure to run the main file on the Gomoku Server folder (project) first, so that the code from Gomoku Client repo (project) could run properly.

Gomoku Client repository: [link](https://github.com/HwuPhuc/GomokuClient)  
Introduction about Gomoku Online Multiplayer Game: [link](https://github.com/HwuPhuc/gomoku-online-multiplayer-game)
